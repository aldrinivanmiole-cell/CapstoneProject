### вң… CAN DO:
- Navigate backwards within same difficulty
- Review answers in current difficulty
- Auto-save on every action
- Recover progress after interruption
- See difficulty badges
- Use confirmation before submitting answer

### вқҢ CANNOT DO:
- Go back to previous difficulty levels
- Skip questions without answering
- See results before completion
- Bypass confirmation (for MC/Yes-No)
- Change answers across difficulty boundaries

### рҹ”’ LOCKING BEHAVIOR:
- Locks happen when moving to NEW difficulty
- Within same difficulty: NO LOCKS
- Once locked: PERMANENT until submission
- Visual indicator: Grayed out navigation dots

### рҹ’ҫ AUTO-SAVE TRIGGERS:
- Answer input changes
- Confirmation acceptance
- Navigation between questions
- Page visibility changes
- Before page unload
